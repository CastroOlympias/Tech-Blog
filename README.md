## Installation
Install in your terminal:
  <br>
* npm install dotenv
  <br>
* npm install express
  <br>
* npm install Node.js
  <br>
* npm install sequelize
  <br>
* npm install npm install --save mysql2
  <br>
* npm install express-handlebars
  <br>
* npm install express-session
  <br>
* npm install bcrypt
  <br>
